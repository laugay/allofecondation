Power analysis of seed viability

Margaux Jullien

05/10/2016

Data simulation

Seed viability data was simulated using the sim.glmm function (package RCurl), specifying Box, Plant, Pod, and Genotype of each seed and the variance of each effect. For Box, Plant and Pod effect, the variance obtained during the analysis of the experimental data was used. For the Genotype effect, 100 simulations were performed specifying variances from 0 to 1.1 (by steps of 0.05). During each simulation, the model Viability ~ (1|Box) + (1|Pod) + (1|Plant) + (1|Genotype) was computed to estimate the genetic variance.
For each simulation, 100 bootstrap estimates were computed to get confidence intervals (CI) of the genetic variance estimated by the model. The 100 CI were used to determine the percentage of simulations with genetic variance significatively different from zero for each value of the specified variance.
